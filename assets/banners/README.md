# Banners do perfil

Coloque suas imagens de banner nesta pasta e cadastre o caminho nos arrays de banners:

- `loja/loja.js` em `ITEMS.banners`
- `perfil/perfil.js` em `PROFILE_BANNERS`

Exemplo de caminho:

```js
image: "../assets/banners/meu-banner.png"
```

Recomendado: imagens horizontais em PNG ou JPG, por exemplo `1200x420`.
